# 🚀 One-Command Setup

## For Windows Users

1. Open the project folder
2. Right-click in empty space → "Open Command Prompt here" (or "Open PowerShell here")
3. Copy and paste this command:

```bash
setup.bat
```

4. Press Enter
5. Browser will open → Login to Vercel with your account
6. Select your project when asked
7. Wait for it to finish

## For Mac/Linux Users

1. Open Terminal
2. Navigate to project folder:
```bash
cd /path/to/Sculptor-of-your-body
```

3. Run:
```bash
bash setup.sh
```

4. Browser will open → Login to Vercel
5. Select your project
6. Wait for completion

## What happens automatically:

✅ Installs Vercel CLI (if not installed)
✅ Logs you into Vercel
✅ Links to your project
✅ Adds TELEGRAM_BOT_TOKEN environment variable
✅ Waits for Vercel to deploy
✅ Registers Telegram webhook
✅ Confirms bot is active

## When it's done:

You'll see: **"🎉 Bot is now active!"**

Test the bot:
https://t.me/MK_sculptor_bot?start=product=program&lang=ru

That's it! No other steps needed. ✨
